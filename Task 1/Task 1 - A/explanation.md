Git & GitHub
Fork :
    -> Creating your own copy of a repositoryon GitHub
    -> It copies the whole project to your GitHub account
    -> This is done so that u can make changes in the project without affecting the original project

Clone:
    -> Downloading the repository from GitHub to your computer

Add:
    -> When u make changes in a file, Git does not track it directly
    -> You must add them using add command

Commit:
    -> This command saves your changes to Git history

Push:
    -> Uploads the changes from your computer to GitHub

Pull:
    -> Pull downloads the latest changes from GitHub to your computer
